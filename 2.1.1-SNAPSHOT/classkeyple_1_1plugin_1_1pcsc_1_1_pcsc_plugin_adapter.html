Detailed Description

(package-private)
Implementation of AbstractPcscPluginAdapter suitable for platforms other than Windows.

Provides a createInstance method that receives a boolean as an argument to indicate that the platform is MacOS.
This information is used to create readers capable of handling the technical issues specific to this platform.

2.0.0
